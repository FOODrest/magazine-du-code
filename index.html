The basic Language of the web: HTML TOP

Posted by Laura-jones on june21st 2027

All moder websites and applicatons are built using three fundamental technologies: HTML, CSS, and JavaScript. These are the lungages of the web.

In this post let's focuse on HTML. We will learn what html is about, and why you shou learn it.

What is HTML ?

HTML stands for HyperText Markup Language. It's a markup language that developers use to describe and structure the content of a webpage (not a programming language).

HTML consists of elements that describes different types of content: paraghrphs, links, heading, images,video,etc. Web browsers understand HTML and render html code as website.

In HTML. Each element is made of 3 parts :

  1. The oppening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs

Why you should learn HTML ?

There are countless reasons for learning the fundamental language of the Web
Here are five of them:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful instead of relying on tools like wordpress or Wix
  • To build web applications
  • To impress friends
  • To have fun😀
